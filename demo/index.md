# Live Demo

The interactive WebAssembly demo is deployed alongside this documentation.

The interactive WebAssembly demo is deployed at `/demo/` on GitHub Pages (built from `FloatPanel.Browser`).

Run locally:

```bash
dotnet run --project FloatPanel.Browser
```

Or publish for static hosting:

```bash
dotnet publish FloatPanel.Browser/FloatPanel.Browser.csproj -c Release
```
